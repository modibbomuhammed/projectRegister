# PMO (Project Management Office) Application

A Node.js/Express application for recording and managing project data with advanced search capabilities.

## Features

- **Project CRUD Operations**: Create, Read, Update, Delete projects
- **Keyword Search**: Search projects across multiple fields (name, description, keywords, tags, etc.)
- **Advanced Search**: Filter projects by status, priority, department, date ranges, and budget
- **Project Statistics**: Get summary statistics of all projects
- **Keyword Suggestions**: Get all unique keywords and tags used across projects
- **REST API**: Fully functional API for integration with frontends
